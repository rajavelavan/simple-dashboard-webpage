# Premium HDPE Pipes & Coils Dashboard - Mangalam Pipes

A high-performance, pixel-perfect dashboard web application for **Mangalam Pipes**, built using vanilla web technologies. This project showcases a modern, responsive interface for an industrial manufacturing company, focusing on product presentation, technical specifications, and lead generation.

## 🚀 Project Overview

This dashboard was developed to provide a premium user experience for clients looking for HDPE piping solutions. It translates a detailed Figma design into a fully functional, responsive webpage with several interactive components and micro-animations.

## ✨ Key Features

### 1. Interactive Product Showcase
- **Hero Image Carousel**: Smooth transitions between product images.
- **Side-by-Side Zoom Preview**: A high-detail zoom feature that appears next to the image on hover (desktop), allowing users to inspect product quality.
- **Dynamic Thumbnails**: Clickable thumbnails to switch the main display image.

### 2. Modern Navigation & Layout
- **Sticky Header**: A smart header that stays at the top of the page when scrolling, providing quick access to navigation and contact actions.
- **Responsive Design**: Fully adaptive layout that works seamlessly across:
  - Desktop (1240px+ max-width)
  - Tablets (iPad/Android tablets)
  - Mobile devices (Smartphones)
- **Breadcrumb Navigation**: Clear pathing for user orientation.

### 3. Technical & Information Sections
- **Technical Specifications**: A clean, responsive table detailing parameter specifications like diameter, pressure ratings, and material density.
- **Manufacturing Process Tabs**: An interactive tabbed interface that walks users through the 8-stage manufacturing process (Raw Material, Extrusion, Cooling, etc.).
- **FAQ Accordion**: A space-efficient way to answer common customer queries with smooth expand/collapse animations.

### 4. Engagement & Lead Generation
- **Applications Carousel**: A sliding carousel displaying various industry use cases for HDPE pipes.
- **Lead Generation Modals**: Functional popups for "Get Custom Quote" and "Download Technical Datasheet" with integrated forms.
- **Newsletter/Catalogue Subscription**: A streamlined form for users to request the full product catalogue via email.
- **Testimonials Section**: A grid of customer success stories highlighting reliability and support.

### 5. Resources & Portfolio
- **Portfolio Grid**: Showcases additional products like HDPE fittings and professional installation services.
- **Resource Downloads**: A dedicated section for technical manuals and specification sheets.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3 (Vanilla)**: 
  - Custom Design System using CSS Variables.
  - Flexbox and CSS Grid for complex layouts.
  - Responsive media queries with custom padding logic.
  - Glassmorphism and subtle micro-animations.
- **JavaScript (Vanilla ES6)**:
  - Custom carousel logic.
  - Image zoom implementation.
  - Tabbed interface and accordion logic.
  - Modal state management.
- **Icons**: Font Awesome 6.4.0 and custom high-resolution PNG/SVG icons.
- **Typography**: Google Fonts (Inter and Urbanist) for a professional and modern look.

## 🎨 Design Philosophy

The project follows a **Premium Industrial** aesthetic:
- **Color Palette**: Deep Navy (#1D3557) for authority, Accent Red (#E63946) for actions, and clean white/light-grey backgrounds.
- **Typography**: Focused on hierarchy and readability.
- **Interactions**: Smooth, predictable transitions that enhance the user experience without being distracting.

## 🔧 How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. No build step or dependencies are required as the project uses vanilla technologies.
